2015/12/16

紙に感情を書き出すといいらしい

この日記は、メンヘラ Advent Calendar 2015の16日目のポエムですw

ここんところやる気ないっす。
ここ十日ほどで、メンヘラ的最新情報がなんかあったんだよな。。。

認知行動療法関連の何だっけ、、、w

えーと、忘れた。

うーとですよ、、、。なんか、血流が前頭葉のなにかで、感情をブロックですよ。
なにいってんすかね。。。

そうだ、テストをするときに感情を紙に書き出すと、成績が上がるらしいです。

鬱対策にもいいらしいのだけど、感情に支配されて不安で一杯になったりしてると、辛いし、パフォーマンスを上手く出せません。

感情コントロールしたいと誰もが願うけど、感情に圧倒されて負けてどうしようもない。
風呂にはいろうが何しようが無駄だ。そんな時代もあった訳ですけど、今の時代はそうじゃない。

脳の血流やCTIかなんかで脳の状態を見ながら、脳の状態を把握して見る事が出来る。
すばらしいことです。何と言う支離滅裂な文章だろうw

でもいいんだ。ま、そんな話です。
要するに、日記とか書いて嫌な事とか書き出しておくとそれだけでもいいらしいという事です。
書き出した後は忘れないと大変ですと。

で、そう俺の苦手な物を思い出して、嫌になった。フラッシュバックを起こすと大変ですよね。
ああ、また思い出してしまった。嫌だ、死にテェで一日が過ぎ去って行きます。勿体ない。
ああ、こんなに辛いのに、何も進んではいませんw

これは勿体ないので、嫌だ。嫌でなくなくない。嫌になる。嫌にならなくない体操みたいなのを考えました。

忘れ去る事もまた出来るはずである。怒っている、嫌になっていると自覚するとまずよい。
自覚したら次は、その状態から良い状態に戻る方法を考えると良いらしいです。

俺の場合、銀行の手続きがもう嫌いで思い出すともう嫌で嫌でしかたなくなる時があります。
ああーもう、嫌絶対やりたくない。って思うとねもう駄目っす。

2015/12/6

適当に思う事を書く

この日記は、メンヘラ Advent Calendar 2015の6日目のポエムですw

恋愛やらオタサーの姫がネタに上がってるようですけど、別に気持ち悪くないじゃんって思ったりする。
姫と呼ばれていた事もあった。だが俺は男であり、決して姫という見た目ではない。なぜに俺は姫と呼ばれたのかというと
オタサーの姫だったらしい。オタサーの姫に振り回されたりした事もあるなとか。でもまぁ、楽しかったなw

ま、共依存もいいさ。うまくいけば。でも依存し過ぎで良くない事があればなくしたほうがいい。当たり前だ。

いろいろメンヘラについては思う事があるんですが、いろいろみんな悩みがあります。
本当に人それぞれ症状が違います。
ただ、他の人の影響を受けやすい人も多いようです。特に、躁鬱病というか今は双極性障害とか言うみたいですけど、アップダウンが激しい人は調子いいときに凄く調子いいので、こうするといいああするといいっていう事になりがちらしいです。

自分もまぁそういう所があります。でもまぁ言いたい事は言いたい訳です。

とりあえず、メンヘラの人はメンヘラの人同士だと安心です。でも、そこに安住ってなると人間になりたいけどなれないみたいになりがちです。なので、まぁ、ある程度症状がよくなったら普通に暮らしたくなります。

自信を持ちましょう。

鬱な人は、大抵、大体みんなまじめ過ぎです。そして、自己評価が低すぎる事が多い。自己評価が低過ぎると人と会うのが億劫になったり出かけるのが面倒になったり怖くなったりする。なので、自信を持ったほうがいいです。
見た目にコンプレックスがあるなら、コンプレックスを持ちすぎないほうがいい。

そんなに見た目が酷い人は、そうはいません。確かに、本当に見た目がよくない人がいたりします。例えば痣があったり、太っていたり、禿げてたりね。そういう人は自信が持てない事でしょう。そこで、うじうじしてる感じが一番かっこ悪いと思うのです。

開き直ったほうがサッパリしていて気持ちがいいです。ハゲの人がネタとして、帽子を被り、おはようございます！って帽子を脱ぐ。
これでもう、ハゲのコンプレックスは解消です。向き合いかたが大事です。

自信を持ちすぎないようにしましょう。

たまに、そうか、自信もてばいいのか！っていって自信満々になってしまう人がいます。これは危険です。
当たり前なのですが、自信はありすぎても良くないし、無さすぎても良くないのです。バランスが大事です。

考えすぎないようにしましょう。

大概、悩み過ぎです。同じ事を堂々巡り。やめましょう。
そんなこといっても、考えないと解決出来ないだろう！そう思うかもしれませんけど、鬱なら考え過ぎです。

考えなさすぎないようにしましょう。

そううつになるパターンとして、そうか！考えすぎなきゃいいんだ！って、考えなくなるとこれまた危険です。
考え過ぎもよくないけど、考えなさ過ぎも問題です。薬飲んでて考えなくなると、下手すると吹っ飛びます。

幻聴や、幻覚などにビックリしない。

薬飲むってことは、上手く叩いていればいいんですが、あわない薬を飲むと色々、おかしなことが起きます。
体が重くて動けなくなってみたり、調子良くなりすぎたり。特に慣れないうちは、ビックリしておかしな妄想に捕われてしまいかねません。
そこは注意したほうがいいです。

薬以外の影響も考えよう。

コーヒー等のカフェインは気分が上がります。これと薬を組み合わせれば上がりすぎかねません。
その辺あまり説明してくれない先生もいます。それなりに調べたり聞くと良いです。

いきてれば良い事あるさ

死にたいって思うけど、死にたくない。よくあるパターンですね。それでいいんです。
死にたいって思いたくなるほど、何か嫌な事がある。そこから逃げ出したい。
って重いが死にたいっていう感情の正体です。
でも、死にたくないから自殺しないんです。
自殺しても怖いから死ぬに至らない。それでいいんです。
出来れば自殺しないほうがいい。自殺するくらいなら、嫌な事を忘れましょう。

でも、死ぬまで解消出来ない辛さも場合によってはあります。
辛いですねぇ。そいつぁしかたないっつ。

やるときはやる

嫌な事をずっと忘れてばかり入られない事もあります。
やるときはやりましょう。悩まず、やる。そして、嫌な事はさっさと忘れる。
嫌な事で頭がいっぱい、だけどやってない。ってのが一番良くないです。

コンフリクトを見つけろ！

大体の悩みには、ジレンマがだいたいあります。
痩せたい。食いたい。食いたいけど痩せたい。だから悩む。
くいたいでも太る。くいたいでも太る。っくいたいと思えば食いたくなる。
辛いです。無意識下でもこのようなことが起きる。食いたいときにちょっと食べる。食い過ぎない。
我慢マックスになると反動がある。ゆっくり楽しんで食べましょう。

信じる物は救われる

薬が聞くと思えば、プラシーボ効果で、ただの歯磨き粉でも聞いたりします。
先生を信じていてうまくいっていればそれでいいんです。
疑い始めると、良くない事が多いだから信じると良いです。
コーヒー飲んだら眼がさえる！って思う。といいです。

信じすぎない

当たり前ですけど、世の中には嘘つきもいます。何でもかんでも信じちゃ駄目です。
他の人の言う事は参考にしつつ、自分なりに判断する事が大事です。
他の人の言う事も聞いたり、考えてみたりして検証が必要です。

宗教も悪くもないよ

これも人によりますよねぇ。妄信は禁物だけど、宗教も場合によっては悩みの解決になります。

相談しよう

相談出来る人がいれば、相談しましょう。悩みを人に言えないと辛いです。
誰でもいいから相談出来れば、解決の第一歩になります。

相談出来ねーよ

言い辛い悩みは、相談出来ないから辛いんだよなぁ。
思い切って相談してみたらいいと思うけど、出来ないならしかたない。分かるよ。
勇気出して相談してみようって言われてもな。言えないんだわな。

愚痴も悪くないよ

溜まった鬱憤、掃き出せば楽になる。

愚痴り過ぎは良くない

愚痴ると楽だけど、愚痴ってばかりであると、それが周りに悪影響を及ぼす。
それが負のフィードバックになりかねない。愚痴り過ぎは良くない。

ただの肩こりかもよ。。。

うつになるとダルいって症状が出たりします。
緊張すると固くなります。固くなれば、肩もこります。肩が凝れば血行が悪くもなります。
血行がだるくもなります。

運動すると眠れるよ

眠れない人が場合によっては運動したら疲れて眠れるようになります。
俺は、そうでした。

運動しても寝れない人もいる

運動しなければ！ってくたくたになった。だけど眠れない。そういってる人がいました。
運動しても眠れない。

運動しなければ！って思うのは自己暗示になるし、運動したのに眠れないって思うのも自己暗示になる。
催眠効果があるのでそう思うのは良くないです。でも、原因が自己暗示による物だけではないかもしれません。
体が疲れて、頭が疲れて、体温がいい感じに上がったり下がったりして、暗くなって、気持ち良くなってうとうとして寝るわけですけど、なにかしらのバランスが崩れてるから眠れないのです。

薬飲んでも眠れない

薬飲んでも寝ようとしないと眠れません。考え事をすると眠れません。眠れない眠れないって意識すると眠れなくなります。
無意識に寝れるのが一番です。

薬飲めば眠れる。

条件反射的に眠れたらそれがいちばん。薬飲まずに寝れたら最高。

2015/12/2

gitの一部だけ取り出してリポジトリを作る

今、docsというリポジトリがあります。
この中から一部分(ocaml/jasc)というフォルダだけ取り出して、新しいgitリポジトリを作りたいと思います。

git clone docs jasc
cd jasc
git filter-branch --subdirectory-filter ocaml/jasc HEAD

git filter-branchを使う事でocaml/jascがルートディレクトリになり、ocaml/jasc内の変更履歴のみが残ります。
理由は分かりませんが、cloneのほうがcp -rfするよりも高速でした。
